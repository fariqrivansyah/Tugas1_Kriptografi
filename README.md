Aplikasi web kriptografi klasik berbasis Flask dengan tampilan modern cyber security dashboard.  
Project ini mendukung berbagai algoritma enkripsi dan dekripsi lengkap dengan visualisasi proses perhitungan.

---

# 🚀 Features

## ✅ Supported Algorithms

- Caesar Cipher
- Vigenère Cipher
- Affine Cipher
- Hill Cipher
- Playfair Cipher

---

# ✨ Main Features

- Encrypt & Decrypt Text
- Step-by-Step Cryptography Process
- Mathematical Formula Visualization
- Hill Cipher Matrix Visualization
- Playfair 5x5 Matrix Visualization
- Encryption & Decryption History
- Dark / Light Mode
- Responsive UI
- Cyber Security Dashboard Design

---
